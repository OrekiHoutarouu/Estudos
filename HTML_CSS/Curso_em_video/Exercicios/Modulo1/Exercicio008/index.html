Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica)

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica)

Texto marcado

Nesta frase, temos um termo marcado usando a tag <mark> e mudando a cor com atributo style

Nesta frase, temos um termo marcado usando a tag <mark> que não puxa o atributo da de cima

Texto grande e pequeno

Nesta frase, Estamos criando um texto grande e um texto pequeno com as tags <big> (não semântica) e <small> (semântica)

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado com a tag <del>

Texto inserido

Podemos marcar um texto como incluído para dar uma ênfase que ele foi adicionado depois com a tag <ins>

Existe também o sublinhado com a tag <u> (que é não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+ 3 com a tag <sup>

Texto subescrito

Para inserir coisas do tipo H2O com a tag <sub>

Curiosidade

Se o atributo style for colocado na tag <head> com o seletor mark, todas as tags <mark> da tag <body> serão modificadas para o que foi configurado na tag <style>
